# Stack tecnica

## Frontend

- Next.js 16 App Router.
- React.
- Tailwind CSS.
- shadcn base.
- lucide-react para icones.

## Backend

- Next.js API Routes.
- TypeScript.
- Zod para validacao de payloads.
- Supabase JS client.

## Banco e auth

- Supabase Auth.
- Supabase Postgres.
- Row Level Security.
- SQL migrations.
- Funcoes PL/pgSQL para RPCs e regras.

## Infraestrutura

- Vercel para deploy do CRM.
- Vercel para landing page.
- Supabase remoto como backend compartilhado.

## Automacao

- n8n via Webhook, HTTP Request e Schedule Trigger.

## Repositorios

- CRM: `crm-ai-native`.
- Landing: `synclead`.
