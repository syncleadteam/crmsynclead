# Design system

## Direcao visual

O CRM segue a identidade visual da landing SyncLead: interface escura, acentos roxo/ciano, logo no topo e linguagem operacional de agencia de automacoes.

## Cores

Tokens principais definidos em `src/app/globals.css`:

- Background: `oklch(0.13 0.012 270)`
- Foreground: `oklch(0.985 0.002 247)`
- Card: `oklch(0.17 0.012 270)`
- Primary: `oklch(0.52 0.28 295)`
- Primary glow: `oklch(0.65 0.30 295)`
- Accent cyan: `oklch(0.84 0.16 220)`
- Muted: `oklch(0.22 0.012 270)`
- Border: `oklch(1 0 0 / 8%)`
- Destructive: `oklch(0.7 0.19 22)`

## Tipografia

- Fonte principal: Geist Sans.
- Fonte mono: Geist Mono.
- Titulos: semibold, tracking normal, evitando hero text dentro do CRM.
- Labels: pequenos, objetivos e com cor muted.

## Espacamento

- Padding de pagina: `px-4 py-6`, com `lg:px-8`.
- Cards e paineis: `p-4` ou `p-6`.
- Gaps de grids: `gap-4` a `gap-6`.
- Border radius base: `0.75rem`.

## Linguagem

Termos preferidos:

- Funil em vez de pipeline.
- Oportunidade em vez de deal.
- Conta em vez de empresa.
- Consultor em vez de owner.
- Agenda em vez de tarefas.
- Automacoes n8n em vez de integracoes genericas.
