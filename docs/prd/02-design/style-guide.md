# Style guide

## Tom de voz

O CRM deve falar como uma agencia de automacoes: direto, operacional, consultivo e sem jargoes tecnicos desnecessarios.

## Padroes de texto

- Use frases curtas.
- Evite termos em ingles quando houver termo operacional claro.
- Use "automacao", "oportunidade", "conta", "consultor", "funil" e "agenda".
- Use "n8n" apenas quando a automacao externa for explicitamente a ferramenta.

## Labels

Preferir:

- "Nome"
- "Conta"
- "Contato"
- "Consultor"
- "Valor"
- "Etapa"
- "Status"
- "Ultima sync"

Evitar:

- "Deal"
- "Pipeline"
- "Owner"
- "Provider" quando o contexto for visual.

## Mensagens de erro

Devem explicar a falha em linguagem de usuario:

- "Nao foi possivel listar oportunidades."
- "Oportunidade nao encontrada."
- "Callback N8N nao autorizado."

## Capitalizacao

Usar sentence case em descricoes e labels. Titulos podem usar primeira palavra em maiuscula.
